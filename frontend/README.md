# GitHub Resume Builder - Frontend

React + TypeScript frontend for the GitHub Resume Builder.

## Tech Stack

- React 18
- TypeScript
- Vite
- React Router
- Tailwind CSS
- Axios

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure environment:
```bash
cp .env.example .env
# Edit VITE_API_URL if needed
```

3. Run development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Features

- GitHub OAuth authentication
- Resume generation with AI
- Resume management (list, view, delete)
- Responsive design
- Protected routes

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
